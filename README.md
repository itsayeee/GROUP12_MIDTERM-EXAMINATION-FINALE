  ## 2B. SIGN

        -> The Excel SIGN function returns the sign of a number as +1, 1, or 0.

              =𝘓𝘊𝘔(𝘯𝘶𝘮𝘣𝘦𝘳1, [𝘯𝘶𝘮𝘣𝘦𝘳2], …)
